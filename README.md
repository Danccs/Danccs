Here are some of the repositories and projects I have developed:

*   📦 **[CleanArchitectureTemplate](https://github.com/Danccs/CleanArchitectureTemplate)**: A production-ready solution template for ASP.NET Core, structured under Clean Architecture principles to keep software maintainable and independent of external frameworks.
*   🔗 **[ApiEndpoints](https://github.com/Danccs/ApiEndpoints)**: A project that enables native API Endpoints support in ASP.NET Core to structure controllers modularly (following the REPR pattern: Request-Endpoint-Response).
*   📚 **[DotNetDataAccessTour](https://github.com/Danccs/DotNetDataAccessTour)**: An educational and demonstrative repository exploring in-depth different data access approaches and patterns in the .NET 8+ ecosystem.
*   💼 **[CastorWallet](https://github.com/Danccs/CastorWallet)**: A modern personal finance mobile app oriented towards efficient expense tracking and budget management built with TypeScript.
*   👥 **[DuoFinance](https://github.com/Danccs/DuoFinance)**: A shared finance solution designed for couples to maintain transparent accounts and common financial goals interactively.
*   ⚙️ **[EquiposIT](https://github.com/Danccs/EquiposIT)**: A .NET platform developed for managing, assigning, and inventorying IT hardware and resources in corporate environments.
